# 项目说明

本仓库保存 Word VBA 工程的源码导出：

- `src/Modules`：标准模块（.bas）
- `src/Classes`：类模块（.cls）
- `src/Forms`：窗体（.frm + .frx）
- `src/Documents`：文档模块（.cls）
- `manifest/references.txt`：工程引用清单
